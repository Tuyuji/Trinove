# Trinove

A Wayland compositor library for D, developed for Allium's Leek compositor.

> [!WARNING]
> This is still a work in progress, please look at [Progress](#Progress) for more info.

## Progress

Trinove supports enough protocols to get Wine Wayland and XWayland working, but there 
might be weird behavior or issues.

This is mainly a learning project for me that has spiraled into something that
could maybe be useful in the future.
