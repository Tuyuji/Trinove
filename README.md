# Trinove

A Wayland compositor inspired by BeOS.

> [!WARNING]
> This is still a work in progress, please look at [Progress](#Progress) for more info.

## XWayland

Trinove will focus on supporting xwayland-satellite as it's simpler to deal with,  
and most applications are moving to Wayland anyway.

We take the same stance as Niri seen [here.](https://github.com/niri-wm/niri/wiki/Xwayland#:~:text=We%27re%20using%20xwayland%2Dsatellite%20rather%20than%20Xwayland%20directly%20because)

## Progress

Trinove supports enough protocols to get Wine Wayland and XWayland working, but there 
might be weird behavior or issues.

This is mainly a learning project for me that has spiraled into something that
could maybe be useful in the future.
